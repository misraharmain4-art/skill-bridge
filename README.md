# SkillBridge 🌉

A Student Skill Exchange Platform for college students.

## Tech Stack
- **Frontend**: React.js + Tailwind CSS (v4)
- **Backend**: Node.js + Express.js
- **Database**: Firebase Firestore
- **Authentication**: Firebase Auth

## Project Structure
- `client/`: React frontend (Vite)
- `server/`: Express backend and Firebase Admin configuration

## Getting Started

### Prerequisites
- Node.js installed

### Setup
1. Clone the repository
2. Install dependencies:
   ```bash
   # Root
   npm install

   # Client
   cd client
   npm install

   # Server
   cd server
   npm install
   ```

3. Configure Firebase:
   - Place your `firebase-admin.json` in the `server/` directory.
   - Update `client/src/firebase-config.js` (if implemented) with your web app keys.

### Running the App
```bash
# Start Backend
cd server
node server.js

# Start Frontend
cd client
npm run dev
```
